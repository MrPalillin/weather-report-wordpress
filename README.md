# weather-report-wordpress
A simple widget that shows a table with a forecast of the next seven days of a destination chosen with an input field.

##Installation instructions
1. Download the repository as a ZIP file.
2. In the Plugins section of Wordpress, go to "Add Plugin", then click on the button "Upload Plugin" and select the ZIP file with the downloaded plugin.
3. In the Installed Plugins section, Activate the Weather Widget plugin.
4. To use the plugin, add the following shortcode to a page of your choice:

	[wp_openmeteo][/wp_openmeteo]
